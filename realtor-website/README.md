# Sun and Sand Realtor Website (Standalone)

This is a fully independent realtor website scaffold for Sun and Sand Realtor and is not connected to the CRM app.

## Files

- `index.html` - main site markup
- `styles.css` - site styling
- `script.js` - menu toggle, listing filters, and contact form behavior

## Run Locally

Option 1:
- Open `index.html` directly in your browser.

Option 2 (recommended):
- From this folder, run:
  - `python3 -m http.server 8080`
- Visit [http://localhost:8080](http://localhost:8080)

## Notes

- All content is currently demo content and can be customized.
- Listing images use external URLs from Unsplash.
